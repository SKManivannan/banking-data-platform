# Real-Time Banking Data Platform

## Overview

This project simulates a real-time banking transaction processing system using Kafka, Spark, and NoSQL technologies.

## Goals

* Build a production-style data pipeline
* Learn streaming with Kafka
* Implement fraud detection logic
* Use NoSQL systems like Cassandra and DynamoDB

## Tech Stack

* Kafka (Streaming)
* Spark (Processing)
* Python (Producer/Consumer)
* AWS (S3, DynamoDB - later)
* Redis (Caching - later)

## Project Structure

* docs/ → Documentation
* producer/ → Kafka producers
* streaming/ → Spark jobs
* storage/ → DB configs
